# Talk App

---

    This application was designed to create and join multiple video conference channel to communicate with many people.

--- 

This application have been created from :
  * Node.js 
  * Express
  * Typescript
  * PostgreSQL
  * Prisma
  * Socket.io
  * Peerjs
  * Morgan
  * Pug
  * Cors
  * bcrypt
  * cookie-parser

Dev dependencies :  
  * Jest  
  * nodemon  
  * ts-node  
  * tsconfig-paths
